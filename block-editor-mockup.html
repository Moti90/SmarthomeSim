🎨 Mockup Features v2

🎯 Triggers

• Tid-baserede triggers (klokkeslæt, dag)

• Sensor triggers (bevægelse, dør/vindue)

• Vejr-baserede triggers

• Bruger-baserede triggers

🔍 Betingelser

• Sammenlign værdier (større end, mindre end)

• Tidsintervaller og datoer

• Kombiner betingelser (og/eller)

• Sensor værdier og tilstande

⚡ Handlinger

• Lampe kontrol (tænd/sluk, dæmper)

• Temperatur kontrol (radiator)

• Ventilator og klima

• Notifikationer og beskeder

🎨 Interface

• Drag & Drop blokke

• Visuelt forbindelsessystem

• Realtids egenskabsredigering

• Scene test og gem funktioner

🧠 Intelligent Tilpasning

• Sensor-specifikke indstillinger

• Automatisk valg af relevante muligheder

• PIR: Aktiveret/Deaktiveret + Følsomhed

• Fugtighed: % niveau + Sammenligning

• Temperatur: °C værdi + Tolerance

📍 Smart Lokationsvalg

• Kun tilgængelige sensorer vises

• PIR: 2 sensorer (Stue, Entre)

• Temperatur: 4 sensorer (alle rum)

• Røgalarm: 5 sensorer (alle rum)

• Fugtighed: 2 sensorer (Køkken, Badeværelse)
